Roadmap

 

A brief look into the milestones achieved by the Bitcoin Nova Community and what's to come.

2017

  • PLANNING OF BITCOIN NOVA

    Major steps towards bitcoin nova as a cryptocurrency.

    October 23, 2017

  • WEBSITE LAUNCH

    Register domain bitcoinnova.org and start creating first website.

    November 2, 2017

  • LAUNCH

    Official website at http://www.bitcoinnova.org.

    Github Repository was created.

    November 2, 2017

  • FIRST BLOCK

    Finally! The first bitcoin nova was mined. Block height: 1.

    November 3, 2017

  • START OF BTN

    New Github Repository was created.

    Official BTN Mining Pool was founded.

    November 12, 2017

  • OFFICIAL BLOCK EXPLORER

    New standalone block explorer: http://explorer.bitcoinnova.org

    November 13, 2017

  • FIRST FAUCET

    First official faucet was launched.

    December 14, 2017

2018

  • DESKTOP WALLET RELEASE

    Desktop Wallet for Windows, Ubuntu and Mac OS X was released.

    March 3, 2018

  • WEB WALLET

    Web Wallet is in development.

    June 1, 2018

  • MOBILE WALLET

    The light Mobile Wallet that will work via remote nodes is in development.

    July 1, 2018

  • INCLUSION OF BITCOIN NOVA IN EXCHANGES

    Inclusion in Next.Exchange, Octaex, C-Cex, Livecoin, TradeOgre, BTC Trade UA, Blaziliex, Abucoins, Coinbe, Coinorigami, Crex24, Binance, Southxchance, Bithumb, HITBTC, Bittrex, Poloniex and Cryptopia.

    IN PROCESS...

2019

  • PAYMENT GATEWAYS

    Developing of Payment Gateways for such an e-commerce system like woo-commerce.

    Design of mobile point of sale terminal for stores.

    March 1, 2019

  • SEARCH FOR UNSATISFACTORY NEEDS FOR FUTURE IMPLEMENTATIONS

    Search for unsatisfactory needs for future implementations.

    November 1, 2019
